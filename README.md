# 📱 AI Recruitment Assistant – Android Application

An **Android application prototype built with Kotlin and Android Studio**, designed to explore how mobile applications could support recruitment processes through structured candidate questionnaires.

This project was developed as an early concept for a recruitment tool where candidates could answer AI-inspired interview questions and recruiters could later analyze the collected information.



## 🚀 Project Overview

The main goal of this project was to create a **mobile interface for collecting candidate responses during a recruitment process**.

The application allows users to complete a questionnaire designed to evaluate aspects such as skills, experience, and professional profile. The responses are then **stored in a database for later analysis by recruiters**.

Although the original idea included advanced AI-based evaluation features, the implemented prototype focuses on the **data collection and storage stage of the recruitment workflow**.



## 🧠 Core Features

### 📋 Candidate Questionnaire
- Interactive questionnaire interface
- Structured questions related to professional experience and skills
- Mobile-friendly user interaction

### 💾 Data Collection and Storage
- Storage of candidate responses in a database
- Structured information ready for later analysis

### 📊 Recruitment Workflow Concept
- Early prototype exploring how mobile apps could assist recruitment processes
- Foundation for potential AI-based evaluation systems



## 🛠 Technologies Used

- **Kotlin**
- **Android Studio**

The application uses several standard mobile capabilities such as:

- Touchscreen interaction
- Wi-Fi connectivity
- Local or remote data storage
